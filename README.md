A **simple sitting on stairs and slabs** plugin, useful in your telling times with your friends!

## When it works?
When a player **rightclicks in a stairs or slabs** (in every type of vanilla material) or via the command **/sit**.

## Where it runs?
It works with every versions of Minecraft over **1.18.2**
